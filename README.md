## The Epoch Chain — Why Offline Simulation Dies

Previous AEGIS beasts could theoretically be simulated offline by anyone with the source code. ACHERON introduces **cross-session state coupling**:

```
epoch_chain[n] = SHA256(epoch_chain[n-1] || transcript_hash || epoch || salt)
```

Every query the attacker makes feeds the epoch chain. Two attackers making different query sequences receive completely different solar radiation, different fissure locations, different dehydration curves. You cannot share results between teams. You cannot replay a session. You cannot simulate without the exact transcript of every previous session.

**Measured: 0/50 match between coupled and offline sessions.** Total divergence.
